# 🌸 Secure Web Application

## 📌 Project Overview

This project is a **Secure Web Application** developed using **Flask (Python)**. It demonstrates secure user authentication and secure coding practices by implementing password hashing, CSRF protection, SQL injection prevention, and user authorization.

---

## 🎯 Objective

To develop a secure web application that provides user registration and login functionality while following secure web development practices.

---

## 🛠 Technologies Used

- Python 3
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-WTF
- Flask-Bcrypt
- SQLite
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

---

## ✨ Features

- User Registration
- User Login
- User Logout
- Password Hashing using Bcrypt
- Password Confirmation Validation
- Password Strength Validation
- Duplicate Username Detection
- Duplicate Email Detection
- Protected User Dashboard
- Responsive User Interface
- Show/Hide Password
- Secure Session Management

## 🔒 Security Features

- Authentication using Flask-Login
- Authorization using Protected Routes
- Password Hashing (Bcrypt)
- CSRF Protection (Flask-WTF)
- SQL Injection Prevention (SQLAlchemy ORM)
- Input Validation
- Secure Password Policies

## 🚀 Installation & Execution

### 1. Clone or Download the Project

```bash
git clone <repository-url>
```

or download the ZIP file.

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
python app.py
```

### 4. Open the Browser

```
http://127.0.0.1:5000
```
